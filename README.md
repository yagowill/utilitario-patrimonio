# Utilitário Patrimônio
Ferramenta web para formatação automática. Desenvolvida para uso interno no setor administrativo, eliminando a necessidade de copiar e formatar dados manualmente.

🔗 **[Acessar aplicação](https://www.yagowill.com.br/utilitario-patrimonio)**

---

## O problema que resolve
Consultar dados de patrimônio exigia abrir a planilha, preencher gerar uma sequência, copiar os valores — e ao colar em outro lugar, os dados vinham separados por espaços em vez de vírgulas, exigindo formatação manual. Com essa ferramenta, qualquer colega consegue copiar e colar na ferramenta e ter os dados no formato certo para pesquisar no sistema, sem precisar ter todo um trabalho manual.

## Funcionalidades

- **Formatação automática** — converte dados copiados da planilha para o formato separado por vírgulas
- **Detecção de duplicatas** — identifica automaticamente entradas repetidas na lista, evitando erros de pesquisa no sistema

## Tecnologias
- HTML, CSS, JavaScript puro e Bootstrap 5
- Deploy na [Vercel](https://vercel.com)

## Como usar
1. Acesse [yagowill.com.br/utilitario-patrimonio](https://www.yagowill.com.br/utilitario-patrimonio)
2. Cole os dados copiados da planilha
3. A ferramenta formata automaticamente e sinaliza qualquer duplicata encontrada

## Rodando localmente
Não requer instalação. Basta abrir o `index.html` no navegador:
```bash
git clone https://github.com/yagowill/utilitario-patrimonio.git
cd utilitario-patrimonio
# abra o index.html no navegador
```
